# Введение

Проект разрабатывается в рамках проектной деятельности и направлен на создание современного образовательного веб-приложения для школьников 7–9 классов.

Планируется реализация интерактивных лабораторных работ по физике, химии и биологии, соответствующих школьной программе.

Главная особенность проекта — интерактивность: пользователь может взаимодействовать с виртуальными объектами и наблюдать результаты экспериментов.

**Цель практики:**
- изучение аналогов в сфере EdTech;
- разработка статического сайта;
- освоение Git и Markdown;
- подготовка итогового отчета.

Практика проходила с 03 февраля 2026 г. по 24 мая 2026 г.

# 1. Общая информация о проекте

## Название проекта:
**«Виртуальная лаборатория»**

## Цель проекта:
Создание веб-приложения с интерактивными лабораторными работами для повышения интереса школьников к естественным наукам.

## Задачи:
1. Анализ аналогов (PhET, Labster, Яндекс Учебник).
2. Запуск проекта локально.
3. Освоение Git и Markdown.
4. Разработка 4 лабораторных работ (HTML, CSS, JavaScript, React).
5. Создание сайта-презентации.
6. Подготовка отчета в GitHub.
