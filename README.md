# Проектная учебная практика

## Общая информация
Данный проект выполнен в рамках проектной (учебной) практики студентом первого курса направления, связанного с информационными технологиями.

Проект представляет собой статический веб-сайт, созданный с использованием HTML и CSS, а также системы контроля версий Git.

## Цель проекта
Освоение базовых инструментов веб-разработки и командной работы с использованием Git, а также формирование навыков создания и структурирования веб-страниц.

## Используемые технологии
- HTML
- CSS
- Git
- Markdown

## Структура проекта
- `index.html` — главная страница сайта
- `about.html` — информация о проекте
- `team.html` — участники проекта
- `journal.html` — журнал выполнения работ
- `resources.html` — полезные ресурсы
- `style.css` — стили оформления сайта
- `images/` — изображения, используемые на сайте

## Результаты работы
В ходе выполнения проекта был создан статический веб-сайт, оформлена структура репозитория, освоены базовые команды Git и принципы работы с Markdown.
